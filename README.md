# Landing Page استشارات إطلاق SaaS

هذه المستودع يحتوي على صفحة هبوط ثابتة باللغة العربية موجهة لرواد الأعمال في السعودية الراغبين في إطلاق منتجات SaaS. الصفحة تبرز الخدمات، المنهجية، القيم المضافة، وآراء العملاء مع دعوات واضحة للحجز عبر واتساب.

## البدء

لا توجد متطلبات تشغيل خاصة؛ كل ما عليك هو فتح ملف `index.html` في المتصفح.

## الملفات الرئيسية

- `index.html`: هيكل الصفحة والأقسام المختلفة.
- `styles.css`: الأنماط والتصميم المتجاوب.
- `script.js`: منطق بسيط لفتح وإغلاق قائمة التنقل على الأجهزة الصغيرة.

يمكنك تحديث رابط واتساب في جميع الملفات إلى رقمك الخاص لضمان وصول العملاء إليك مباشرةً.
